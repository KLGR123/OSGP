# Conducting Training System based on Leap Motion
一个基于leap motion的指挥训练系统
本项目可以为用户的指挥动作进行评分(以2/4拍为例),并且根据用户挥拍的速度实时调整乐谱行进的速度。
#Features
-leap motion
-conducting training system
-gesture recognition
#Requirements
-software: Processing 3.3.6, MuseScore 2.1.0, Wekinator 2.0
-OS: Windows 10
-libraries: Leap Motion for Processing, oscP5

#Install
Install Leap Motion Orion 3.2.1
Install Processing 3.3.6
Install MuseScore 2.1.0
Install Wekinator 2.0

#Usage
-Insert the Leap Motion
-Run tempo.wekproj
-Run point.pde
-Open swan_lake-scene.mscz
-Run leap.pde
